# Hello World — 博客搭建记

终于有自己的博客了！

## 技术栈

这个博客完全由纯前端技术搭建：

- **HTML5** — 语义化页面结构
- **CSS3** — 动画、渐变、Flexbox/Grid 布局
- **JavaScript** — 粒子动画、Markdown 渲染、交互逻辑
- **marked.js** — 实时解析 Markdown 文章
- **highlight.js** — 代码语法高亮

## 为什么不用框架？

> 简单的事情不需要复杂的工具。

对于一个个人博客来说，纯静态页面就够了。没有构建步骤，没有依赖地狱，一个浏览器就能跑。

## 部署

最后通过 GitHub Pages 免费部署，全世界都能访问，手机也完美适配。

```bash
# 部署只需要推送到 GitHub
git push origin main
```
